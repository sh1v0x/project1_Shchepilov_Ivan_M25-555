# project1_Shchepilov_Ivan_M25-555
## Лабиринт сокровищ
 
Лабиринт сокровищ - это игра, в которой игрок, используя команды, исследует комнаты с целью найти клад. В игровом процессе пользователь также взаимодействует с различными предметами, решает загадки, уворачивается от ловушек.

## Установка
1. Для  начала необходимо клонировать репозиторий:
```
git clone git@github.com:sh1v0x/project1_Shchepilov_Ivan_M25-555.git
cd project1_Shchepilov_Ivan_M25-555
```
2. Необходимо установить зависимости
```
poetry install
# или
make install
```
## Запуск игры
```
make project
```
